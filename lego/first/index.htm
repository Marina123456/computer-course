1.2   Скачиваем

Загрузи программное обеспечение (ПК/МАС)

Нажмите Download for Windows

2.       Установи дополнительные блоки для датчиков. Они устанавливаются отдельноL

2.1. Перейди на сайт https://www.lego.com/ru-ru/mindstorms/downloads

Скачай два датчика: гироскопический датчик + ультразвуковой датчик

2.2 Подключи блоки датчиков к программе

Tools->Block Import

2.3 Открывается окно

Выбираешь  файлы, которые только что скачал и нажимаешь Import–>выбираем один датчик нажимаем импорт-> выбираем другой датчик нажимаем импорт.

И перезагрузи программу.

 

Установка завершена. Программа настроена. Можешь создавать свои программы.
